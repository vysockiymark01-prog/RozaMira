# Роза Мира — сайт-читалка

Полный текст книги Даниила Андреева **«Роза Мира»** (1950–1958) в виде удобного
сайта для чтения — с пояснениями простым языком, наглядными схемами и поиском.

«Роза Мира» — философско-мистический трактат, написанный поэтом Даниилом
Андреевым в основном во Владимирской тюрьме. Книга сложна для неподготовленного
читателя: сотни авторских терминов, многослойная картина мира, необычный язык.
Этот проект делает её доступнее для обычного человека.

## Что внутри

**Текст книги** — все 12 книг и 43 главы, разбитые по страницам, с навигацией
«вперёд/назад», оглавлением и хлебными крошками. Перед каждой книгой и каждой
главой — краткое содержание простым языком.

**Карта миров** — 9 интерактивных схем по содержанию книги с переключением
вкладками: слои Шаданакара, посмертный путь души, строение человека, затомисы,
иерархии света, тёмные силы, стихиали, вестники русской культуры, смена эонов.
В исторических книгах (VII–XI) — лента метаистории России.

**Справочные разделы:**
- словарь терминов (авторский словарь + пояснения составителя) со схемой связей понятий;
- «Кто есть кто» — карточки главных светлых и тёмных сущностей книги;
- примечания, страница об авторе и советы, с чего начать чтение.

**Удобства чтения:** светлая и тёмная темы, регулировка размера шрифта,
полнотекстовый поиск по всем главам, запоминание места чтения, отметки
прочитанных книг, кнопка «наверх» и возврат к книге со страниц схем.
Всё хранится в браузере читателя, сервер не нужен.

## Технически

Статический сайт: чистые HTML, CSS и JavaScript, без фреймворков и сборки.
Термины в тексте подсвечены всплывающими подсказками со ссылками на словарь.
Схемы — встроенный SVG, автоматически перекрашиваются в тёмной теме.
Работает локально (открыть `index.html`) и на любом статическом хостинге.

## Структура

```
index.html        — главное меню: 12 книг с аннотациями, маршрут чтения
about.html        — об авторе и книге
map.html          — карта миров (9 схем)
who.html          — кто есть кто
glossary.html     — словарь терминов
notes.html        — примечания
search.html       — поиск по тексту
Главы/            — текст книги: «Книга N — главы X-Y/index.html»
assets/           — стили, скрипты, иконка, поисковый индекс
```

## Публикация на GitHub Pages

1. Загрузите содержимое папки в репозиторий (файл `.nojekyll` обязателен).
2. **Settings → Pages → Source: Deploy from a branch**, ветка `main`, папка `/ (root)`.
3. Через пару минут сайт откроется по адресу `https://<логин>.github.io/<репозиторий>/`.
